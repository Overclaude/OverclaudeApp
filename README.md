# Overclaude

A Windows desktop app for managing multiple Claude Code sessions. Launch instances the way you want, keep them visually organized with color coding, and monitor everything from a single dashboard.

**[overclaude.app](https://overclaude.app)**

> Overclaude is an independent tool and is not affiliated with, endorsed by, or associated with Anthropic or Claude Code.

## Download

Get the latest version from the [Releases](../../releases/latest) page.

## Requirements

- Windows 10 or 11 (64-bit)
- [Claude Code](https://claude.ai/code) installed and authenticated
- WebView2 Runtime (pre-installed on most Windows 10/11 systems)

## Installation

1. Download `Overclaude.zip` from the latest release
2. Extract to a folder of your choice
3. Run `Overclaude.exe`

No installer required.

## Features

- **Profiles** - reusable launch templates with working directory, color, model, monitor, and position settings
- **Color coding** - each instance gets a color that follows it across the dashboard, frame border, and title bar
- **Live monitoring** - see state, cost estimates, context usage, and last action for every running instance in real time
- **Activity tray** - per-turn tool call history with cost and token breakdowns
- **Multi-monitor support** - assign profiles to monitors, view a live monitor map, move instances with one click
